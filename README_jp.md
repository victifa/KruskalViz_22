# KruskalViz

🌐 [中文](README.md) / [English ](README_en.md)/ [日本語（にほんご）](README_jp.md)🌐

クラスカルアルゴリズム（最小生成木）の可視化デモツール。Vue3 を使用して構築されており、ノードの動的追加、辺の重み編集、最小生成木の生成過程のデモが可能です。アルゴリズム教育や学習、研究の参考に最適です。

## プロジェクト概要

グラフアルゴリズム、特に最小生成木アルゴリズムを学ぶ際、直感的でインタラクティブなデモツールが不足しがちです。KruskalViz はこのギャップを埋めることを目指しています：

- ユーザーは可視化されたインターフェースを通じて、**ノードの動的追加／削除**、**辺とその重みの追加・編集**が可能です。
- 手動操作モードと自動デモモードをサポート：Kruskal がどのように辺を選択し、連結成分を統合し、最終的に最小生成木を作るかをステップごとに表示します。
- ユーザーフレンドリーなインターフェース、読みやすいコードで、教育、授業デモ、学生の自主練習や研究デモに適しています。

## 主な特徴

- **ノードの動的操作**：キャンバスをクリックしてノードを作成し、ドラッグで位置を変更可能。
- **辺と重みの編集**：ノード間に辺を描画し、ダブルクリックまたは入力ボックスで重みを変更可能。
- **アルゴリズムアニメーションデモ**：「デモ開始」ボタンをクリックすると、辺のソート、選択、スキップ、集合の統合など、Kruskal アルゴリズム全過程を視覚的に表示。
- **手動／自動切替**：デモを一時停止、ステップ実行、または自動再生速度の設定が可能。
- **状態表示**：現在処理中の辺、選択済みの辺、スキップされた辺、統合操作などを視覚的に表示。
- **リセット／再編集**：いつでもキャンバスをクリアしたり操作を取り消して、グラフ入力を再編集可能。

## クイックスタート

### 事前準備

- Node.js のインストール（推奨 v16 以上）
- npm や pnpm などのパッケージマネージャーのインストール
- Vue 3 プロジェクト構造の基礎知識

### インストールと実行

1. リポジトリをクローン

   ```bash
   git clone git@github.com:victifa/KruskalViz_22.git
   cd KruskalViz_22

1. 依存関係をインストール（`pnpm` の例）

   ```
   pnpm install
   ```

2. 開発モードで実行

   ```
   pnpm run dev
   ```

3. ブラウザでアクセス

   `http://localhost:3000`（またはターミナルに表示されるアドレス）を開くと画面が表示されます。

### ビルドと公開

```
pnpm build
# ビルド結果は通常 dist/ ディレクトリに出力され、任意の静的サーバーや GitHub Pages にデプロイ可能
```

## 使用ガイド

### インターフェース説明

- 左／上部ツールバー：「ノード追加」「辺追加」「削除」「リセット」「デモ開始」などのボタンを含む。
- メインキャンバス領域：ノード（点）と辺（線＋重み）を表示。
- 右／下部ステータスバー：アルゴリズムの現在ステップ、選択された辺リスト、生成済み木の辺、スキップされた辺などを表示。

### 操作手順

1. 「ノード追加」ボタンをクリックし、キャンバスの任意の位置でノードを作成。
2. 「辺追加」モードに切り替え、2つのノード間に辺を作成。
3. 辺の重みテキストをダブルクリック、またはプロパティパネルの入力ボックスで重みを変更。
4. グラフ構築完了後、「デモ開始」ボタンをクリック。
5. デモ中は「一時停止／再開」ボタンや「ステップ実行」モードで進行を制御。
6. デモ終了後、生成された最小生成木を確認、または「リセット」をクリックして再編集。

### アルゴリズムデモの流れ

- システムはまず全ての辺を読み込み、重みの小さい順にソート。
- 連結成分判定に並查集合（または同等の構造）を使用。
- 異なる集合をつなぐ場合は生成木に追加し、集合を統合。それ以外はスキップ。
- すべてのノードが連結されるか、選択可能な辺がなくなるまで繰り返す。
- 画面上では、「現在処理中の辺」をハイライト、「選択済みの辺」は緑、「スキップされた辺」は灰色で表示。
- 最終的な生成木は太線や他の色で強調表示。
