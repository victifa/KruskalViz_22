# KruskalViz

🌐 [简体中文](README.md) / [繁體中文](README_hant.md) / [English](README_en.md) / [日本語](README_jp.md) / [한국어](README_ko.md)🌐

克魯斯卡爾演算法（最小生成樹）可視化演示工具，使用 Vue3 構建，支援動態添加節點、編輯邊權值、演示最小生成樹生成過程。適合演算法教學、學習與研究參考。

## 專案簡介

在學習圖演算法、尤其是最小生成樹演算法時，通常缺乏直觀、可互動的演示工具。KruskalViz 致力於填補這一空白：

- 使用者可以透過可視化介面 **動態添加/刪除節點**、**新增/編輯邊與其權值**。
- 支援手動操作與自動演示兩種模式：展示 Kruskal 如何一步步選邊、合併連通分量、直到生成最小生成樹。
- 介面友好、程式碼清晰、適於教學、課堂演示、學生自主練習或研究演示使用。

## 核心特性

- **節點動態操作**：點擊畫布即可建立節點，拖曳可移動節點位置。
- **邊與權值編輯**：可在節點之間繪製邊，雙擊或輸入框修改權值。
- **演算法動畫演示**：點擊「開始演示」即可觀看 Kruskal 演算法全過程，包含邊排序、選擇、跳過、並查集合併等過程。
- **手動/自動切換**：演示可暫停、單步執行，也可設定自動播放速度。
- **狀態提示**：當前正在處理的邊、已選邊、已跳過邊、合併操作等均有視覺提示。
- **重置/重新編輯**：可隨時清空畫布或回退操作，重新編輯圖輸入。

## 快速開始

### 在此之前

- 安裝 Node.js（建議 v16 或以上）
- 安裝套件管理工具，如 npm 或 pnpm
- 基礎了解 Vue 3 專案結構

### 安裝與運行

1. 克隆倉庫

   ```bash
   git clone git@github.com:victifa/KruskalViz_22.git
   cd KruskalViz_22

1. 安裝依賴（以 `pnpm` 為例）

   ```
   pnpm install
   ```

2. 使用開發模式運行

   ```
   pnpm run dev
   ```

3. 網頁訪問

   打開瀏覽器訪問 `http://localhost:3000`（或終端提示的地址）即可看到介面。

### 构建發布

```
pnpm build
# 構建結果通常在 dist/ 目錄，可將其部署至任意靜態伺服器或 GitHub Pages
```

## 使用指南

### 介面說明

- 左側/頂部工具欄：包含「新增節點」「新增邊」「刪除」「重置」「開始演示」等按鈕。
- 主畫布區域：用於展示節點（圓點）與邊（線段 + 權值）。
- 右側/底部狀態欄：顯示演算法當前步驟、選邊列表、已生成樹的邊、跳過的邊等。

### 操作流程

1. 點擊「新增節點」按鈕，在畫布任意位置點擊建立節點。
2. 切換至「新增邊」模式，點擊兩個節點間建立一條邊。
3. 雙擊邊權值文本或在屬性面板輸入框修改權值。
4. 完成圖構建後，點擊「開始演示」按鈕。
5. 在演示過程中，可透過「暫停／繼續」按鈕或「單步執行」模式控制流程。
6. 演示結束後，查看生成的最小生成樹，或點擊「重置」重新編輯輸入。

### 演算法演示流程

- 系統首先讀取所有邊並按權值從小到大排序。
- 使用並查集（或等效結構）判斷每條邊是否連接兩個不同集合。
- 如果連接，則選入生成樹並合併集合；否則跳過。
- 重複直到節點全部連通或無更多可選邊。
- 在畫面中，將「當前處理的邊」高亮、「已選邊」顯示為綠色、「跳過邊」顯示為灰色。
- 最終生成樹用粗線或其他顏色突出顯示。
