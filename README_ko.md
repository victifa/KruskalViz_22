# KruskalViz

🌐 [简体中文](README.md) / [繁體中文](README_hant.md) / [English](README_en.md) / [日本語](README_jp.md) / [한국어](README_ko.md)🌐

크루스칼 알고리즘(최소 신장 트리) 시각화 도구입니다. Vue3로 제작되었으며, 동적으로 노드를 추가하고, 간선 가중치를 편집하며, 최소 신장 트리 생성 과정을 시연할 수 있습니다. 알고리즘 교육, 학습 및 연구 참고용으로 적합합니다.

## 프로젝트 소개

그래프 알고리즘, 특히 최소 신장 트리를 학습할 때 직관적이고 상호작용 가능한 시연 도구가 부족한 경우가 많습니다. KruskalViz는 이 공백을 채우기 위해 만들어졌습니다:

- 사용자는 시각화 인터페이스를 통해 **노드 동적 추가/삭제**, **간선 및 가중치 편집**이 가능합니다.
- 수동 조작과 자동 시연 두 가지 모드를 지원합니다: Kruskal 알고리즘이 단계별로 간선을 선택하고, 연결 요소를 병합하며, 최소 신장 트리를 완성하는 과정을 보여줍니다.
- 친숙한 인터페이스와 깔끔한 코드 구조로, 교육, 수업 시연, 학생 자율 연습 또는 연구 시연에 적합합니다.

## 핵심 기능

- **노드 동적 조작**: 캔버스를 클릭하여 노드를 생성하고, 드래그하여 위치를 이동할 수 있습니다.
- **간선 및 가중치 편집**: 노드 간 간선을 그릴 수 있으며, 더블 클릭이나 입력창을 통해 가중치를 수정할 수 있습니다.
- **알고리즘 애니메이션 시연**: 「시연 시작」을 클릭하면 간선 정렬, 선택, 건너뛰기, 집합 병합 등의 Kruskal 알고리즘 전체 과정을 볼 수 있습니다.
- **수동/자동 전환**: 시연을 일시정지, 단계 실행하거나 자동 재생 속도를 설정할 수 있습니다.
- **상태 표시**: 현재 처리 중인 간선, 선택된 간선, 건너뛴 간선, 병합 작업 등이 시각적으로 표시됩니다.
- **리셋/재편집**: 언제든 캔버스를 초기화하거나 이전 작업을 되돌려 그래프를 다시 편집할 수 있습니다.

## 빠른 시작

### 사전 준비

- Node.js 설치(권장 v16 이상)
- npm 또는 pnpm과 같은 패키지 관리 도구 설치
- Vue 3 프로젝트 구조에 대한 기본 이해

### 설치 및 실행

1. 저장소 클론

   ```bash
   git clone git@github.com:victifa/KruskalViz_22.git
   cd KruskalViz_22

1. 의존성 설치 (`pnpm` 기준)

   ```
   pnpm install
   ```

2. 개발 모드 실행

   ```
   pnpm run dev
   ```

3. 웹 브라우저에서 접속

   브라우저에서 `http://localhost:3000` (또는 터미널에 표시된 주소)로 접속하면 인터페이스 확인 가능.

### 빌드 및 배포

```
pnpm build
# 빌드 결과는 일반적으로 dist/ 디렉토리에 생성되며, 이를 정적 서버나 GitHub Pages에 배포 가능
```

## 사용 가이드

### 인터페이스 설명

- 좌측/상단 툴바: 「노드 추가」「간선 추가」「삭제」「리셋」「시연 시작」 버튼 포함
- 메인 캔버스 영역: 노드(원)와 간선(선 + 가중치) 표시
- 우측/하단 상태 표시줄: 알고리즘 현재 단계, 선택된 간선 목록, 생성된 트리의 간선, 건너뛴 간선 등 표시

### 작업 흐름

1. “노드 추가” 버튼 클릭 후, 캔버스 원하는 위치를 클릭하여 노드 생성
2. “간선 추가” 모드로 전환 후, 두 노드 사이에 간선 생성
3. 간선 가중치 텍스트 더블 클릭 또는 속성 패널 입력창에서 가중치 수정
4. 그래프 구성 완료 후, “시연 시작” 버튼 클릭
5. 시연 중 “일시정지/계속” 버튼 또는 “단계 실행” 모드로 진행 제어
6. 시연 종료 후, 생성된 최소 신장 트리 확인 또는 “리셋”으로 입력 재편집

### 알고리즘 시연 과정

- 시스템은 모든 간선을 읽고 가중치 순으로 정렬
- 집합(Union-Find) 구조로 각 간선이 다른 집합을 연결하는지 확인
- 연결 시 생성 트리에 포함하고 집합 병합, 그렇지 않으면 건너뛰기
- 노드가 모두 연결되거나 선택 가능한 간선이 없을 때까지 반복
- 화면에서는 “현재 처리 간선” 강조, “선택된 간선” 녹색, “건너뛴 간선” 회색 표시
- 최종 생성 트리는 굵은 선 또는 다른 색상으로 강조
